# Sistema de Organização de Estudos

Um sistema simples em Python para gerenciar tarefas de estudo diretamente pelo terminal.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar como concluída
- Editar tarefas
- Deletar tarefas
- Salvamento automático em JSON

## Tecnologias

- Python
- JSON

## Como executar

```bash
py main.py
